.. _documentapi:
Document api
==========================================================================
.. toctree::
   :maxdepth: 3

Binnen deze architectuur gebruiken we een Document Management Systeem waarvan de leverancier ondersteuning voor de `document api <https://github.com/vng-Realisatie/documenten-api>`_ heeft ingebouwd. Het berichtverkeer voldoet dus aan het formaat van de referentie implementatie maar de implementatie is gedaan op een ander DMS.
